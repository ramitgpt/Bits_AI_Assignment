1. Install WebStorm2023.2.5
2. copy chat-gpt-ramit folder in WebstormProjects folder (C:/users/)
3. Open two terminals in Webstorm.
4. In Terminal 1: command npm run start:frontend
5. In Terminal 2: command npm run start:backend
6. In your browser, the chatbot will open at http;//localhost:3000
7. Put your questions in the text area.
8. It will generate the responses.
